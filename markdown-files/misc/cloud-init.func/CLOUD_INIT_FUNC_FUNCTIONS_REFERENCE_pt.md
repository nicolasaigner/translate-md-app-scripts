# Referência de Funções cloud-init.func

Funções de provisionamento de cloud-init e VMs.

## Funções Principais

### generate_cloud_init()

Gera a configuração do cloud-init.

### generate_user_data()

Gera o script de dados do usuário para a VM.

### apply_cloud_init()

Aplica o cloud-init à VM.

### setup_ssh_keys()

Implanta as chaves públicas SSH.

### setup_static_ip()

Configura o IP estático na VM.

### setup_dns()

Configura o DNS para a VM.

### setup_ipv6()

Habilita o IPv6 na VM.
