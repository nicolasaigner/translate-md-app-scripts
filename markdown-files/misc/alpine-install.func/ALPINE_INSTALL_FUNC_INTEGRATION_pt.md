# Guia de Integração do alpine-install.func

Integração do alpine-install.func com fluxos de trabalho de contêineres Alpine.

## Integração Específica para Alpine

Contêineres Alpine usam:

- `apk` em vez de `apt-get`
- `OpenRC` em vez de `systemd`
- Nomes de pacotes específicos do Alpine
