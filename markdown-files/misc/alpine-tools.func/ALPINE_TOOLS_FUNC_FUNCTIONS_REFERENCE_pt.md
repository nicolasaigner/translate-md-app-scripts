# alpine-tools.func Referência de Funções

Funções de instalação de ferramentas específicas do Alpine.

## Funções Principais

### apk_update()

Atualiza as listas de pacotes do Alpine.

### apk_add(PACKAGES)

Instala pacotes do Alpine.

### apk_del(PACKAGES)

Remove pacotes do Alpine.

### add_community_repo()

Habilita o repositório da comunidade Alpine.

### add_testing_repo()

Habilita o repositório de testes do Alpine.

### Funções de Ferramentas do Alpine

- `setup_nodejs()` - Node.js no Alpine
- `setup_php()` - PHP no Alpine
- `setup_mariadb()` - MariaDB no Alpine
- `setup_postgresql()` - PostgreSQL no Alpine
- (+ mais configurações específicas do Alpine)
